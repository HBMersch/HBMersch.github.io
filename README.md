# Handball Club Mersch 75

Statische GitHub-Pages-Basis fuer den Relaunch der Website des Handball Club Mersch 75.

## Struktur

- `index.html`: Startseite mit Club-, Training-, Matchday-, News- und Kontaktbereichen
- `styles.css`: komplettes Layout und responsive Gestaltung
- `script.js`: pflegbare Trainings- und News-Daten fuer die Startseite

## GitHub Pages

Die Seite funktioniert ohne Build-Schritt direkt ueber GitHub Pages. Fuer die Veroeffentlichung reicht es, dieses Repository mit GitHub Pages auf dem Default-Branch zu aktivieren.

## Naechste sinnvolle Schritte

1. Echte Club-Fotos, Logo und Sponsorenmaterial einpflegen.
2. Zusätzliche Unterseiten fuer Teams, Galerie, Impressum und Datenschutz anlegen.
3. News und Termine aus einer externen Datenquelle oder aus Markdown-Dateien speisen.
